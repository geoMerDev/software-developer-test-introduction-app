# "The Musical Lake" - Software Developer Test
A TypeScript solution for the Musical Lake challenge that analyze animal sounds and their musical patterns.


## Problem Description
Animals create songs together. 
Given a sound the program returns the remaining sounds in that song.
  When given "brr", it returns "fiu, cric-cric, brrah".
  When given "birip", it returns "trri-trri, croac".
  When given "plop", it returns "cric-cric, brrah".
  When given "croac" or "brrah", it returns nothing.


## Technical Stack
  Language: TypeScript
  Containerization: Docker
  Runtime: Bun
